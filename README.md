# L1toM2

La plupart de mes cours, projets et révisions de la fac regroupés dans un seul dépôt.

## À propos

Ce dépôt contient mes notes de cours, projets, exercices et ressources pour mes semestres (L1 → M2). Il est organisé par matière et par année/semestre pour faciliter la révision et le partage.

## Structure du dépôt (exemples)

- /cours/       : notes et supports de cours
- /projets/     : projets et exercices réalisés
- /tp/          : travaux pratiques et énoncés
- /ressources/  : outils, scripts, présentations

(La structure exacte peut varier ; parcourez les dossiers pour voir le contenu détaillé.)

## Langages principaux

Selon la composition du dépôt, les langages les plus présents sont :

- HTML (≈ 30%)
- SCSS (≈ 13.8%)
- Hack (≈ 13.4%)
- Java (≈ 13.2%)
- C (≈ 10.6%)
- JavaScript (≈ 4.4%)
- Autres (≈ 14.6%)

## Comment utiliser ce dépôt

- Cloner le dépôt :

  git clone https://github.com/lefaucheuraxel/L1toM2.git

- Parcourir les dossiers pour trouver les cours ou projets qui vous intéressent.
- Ouvrir les fichiers HTML/SCSS/Java/etc. avec votre éditeur préféré.
